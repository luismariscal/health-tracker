Health Tracker → GitHub Setup

One-time setup so you can work on index.html from your phone, laptop, or claude.ai when you're away from your PC.

What you get

  • A private GitHub repo containing index.html
  • Any device with a browser can now read/edit the file via claude.ai
  • Desktop and online stay in sync via git push / git pull

Run the setup (one time)

1 Open Git Bash in the project folder.
In File Explorer, navigate to C:\Projects\Health Tracker, right-click inside the folder, choose "Git Bash Here".

2 Run the setup script:

bash setup-github.sh

3 Follow the prompts. The script will:

  1. Install gh (GitHub CLI) via winget if missing
  2. Log you into GitHub via browser (choose: GitHub.com → HTTPS → Login with browser)
  3. Initialize git on the main branch
  4. Set your git identity (using GitHub's noreply email so your real email isn't exposed)
  5. Commit index.html and .gitignore
  6. Create a private GitHub repo named health-tracker and push to it
Note — if gh installs fresh: the script will tell you to close the terminal, open a new Git Bash, and run the script again. That's normal — Windows needs a new shell to see the new PATH.
To change repo name or make it public, edit the top of setup-github.sh before running: 
REPO_NAME="health-tracker"
VISIBILITY="private"   # or "public"

After setup — your new workflow

On your desktop (after each Claude Code session)

git add -A
git commit -m "what I changed"
git push

On your phone, laptop, or anywhere else

  1. Open claude.ai → Projects → New Project
  2. Connect GitHub → pick health-tracker
  3. Chat normally — Claude reads index.html directly from the repo
  4. When Claude gives you an updated version, commit it via GitHub's web editor (pencil icon on any file) — creates a commit straight from the browser

Back at your desk

Before starting a new Claude Code session, pull the latest:

git pull

Alternative: Claude Code on the web

Anthropic has Claude Code running in-browser at claude.ai/code. If your plan includes it, you'll see a Code tab at claude.ai — connect the same health-tracker repo there and Claude can edit and open PRs directly, no copy/paste required.
